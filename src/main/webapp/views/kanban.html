Searching with queries

Append the following to a URL:

?searchWhat=thingToSearchFor ie. search for assignee: ?assignee=ckre

You can use assignee, priority, reporter, project, label

You can add multiples by separating each pair with "&" ie. ?assignee=gsilver&priority=major&label=code-review

The first pair always has "?" at the beginning
